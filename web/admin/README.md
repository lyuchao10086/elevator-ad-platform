# Elevator Admin (frontend)

Minimal Vue 3 + Vite admin template for elevator-ad-platform.

Run:

```bash
cd elevator-ad-platform/web/admin
npm install
npm run dev
```

API proxy is configured via `.env.development`.

Structure:
- `src/api/index.js` axios instance to adapt backend endpoints.
- `src/pages` example pages (Dashboard, DeviceList).
- `src/components` common layout pieces.

Follow coding conventions: ESLint + Element Plus UI components.
